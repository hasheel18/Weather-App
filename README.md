# Weather App⛅

A simple, modern weather web app that shows current weather, local time, and helpful advice for any city using the OpenWeather API.

## Live Demo
🔗 [View Live App](https://hasheel18.github.io/Weather-App/)

## Features🪒
- Search by city name and view current temperature, humidity, and wind speed.
- Dynamic background gradient that changes with weather and local time of the city.
- Local date and time for the searched city.
- “Feels like” temperature with daily minimum and maximum temperature.
- Short advice text based on weather conditions, such as taking an umbrella or staying hydrated.
- Custom UI with weather icons, glassmorphism card design, and small animations like a floating icon and fade-in effect.

## Tech Stack🛞
- HTML5
- CSS3
- JavaScript
- OpenWeather Current Weather API

## How It Works🔧
1. Enter a city name in the search box.
2. The app fetches live weather data from OpenWeather.
3. It displays temperature, humidity, wind speed, feels-like temperature, min/max values, and local time.
4. The background and advice text update based on the weather condition.

## Notes📝
- Make sure your OpenWeather API key is added correctly in the JavaScript file.
- The app works best with a stable internet connection because it uses live API data.
